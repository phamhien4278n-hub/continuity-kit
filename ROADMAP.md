# Roadmap

## v0.1

- Public protocol specification.
- YAML schemas for heartbeat, snapshot, and continuity state.
- Example pre-LLM injection payload.
- Example post-LLM validation policy.
- Codex-lite and companion-agent examples.

## v0.2

- Reference validator CLI.
- JSONL append-only event log format.
- Basic memory promotion tool.
- Example integrations for hook-based agent runtimes.

## v0.3

- Dashboard for reviewing heartbeats, snapshots, validation failures, and memory promotions.
- Provenance graph view.
- Policy configuration presets.

## Commercial Edition Ideas

- Team policy management.
- Audit reports.
- SSO / RBAC.
- Hosted dashboard.
- Enterprise self-hosted package.
- Custom runtime integrations.
