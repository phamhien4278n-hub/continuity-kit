# Contributing

Contributions are welcome for noncommercial collaboration under the project license.

By contributing, you agree that:

- Your contribution can be distributed under the repository license.
- You grant Qianhui a perpetual, worldwide, royalty-free, irrevocable license to use, reproduce, modify, distribute, sublicense, and commercially license your contribution as part of ContinuityKit.
- Commercial use by users still requires a separate commercial license unless Qianhui grants otherwise in writing.
- You will not submit private user data, secrets, or proprietary material without authorization.

For substantial contributions, the project may require a separate Contributor License Agreement (CLA) or Developer Certificate of Origin (DCO).

Good contribution areas:

- Schema improvements.
- Runtime integration examples.
- Validator rules.
- Memory promotion policies.
- Documentation clarifications.

Please avoid:

- Claims that ACP proves AI consciousness.
- Unverifiable subjective-state claims.
- Examples containing private conversations.
- Vendor-specific marketing language without technical detail.
